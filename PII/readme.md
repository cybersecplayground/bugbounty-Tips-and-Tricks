### Description
Personally Identifiable Information (PII) exposure occurs when sensitive data like names, emails, national IDs, phone numbers, or financial information is improperly stored, transmitted, or accessible without authorization. This folder contains practical resources to help you identify, validate, and responsibly report these critical data privacy flaws.
